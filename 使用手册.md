# 个人网站使用手册

## 启动网站
双击桌面/主目录的 **启动个人网站.command**
然后浏览器打开 → http://localhost:3000

---

## 一、修改个人信息

### 侧边栏（名字、邮箱、社交链接）
文件：`components/Sidebar.tsx`
找到对应文字直接改：
```
张宁宁 Ningning Zhang   ← 改名字
统计学博士 · 香港大学    ← 改职位/单位
zhangnn0725@163.com     ← 改邮箱
```

### 关于我页面（简介、工作/实习/教育经历）
文件：`app/[locale]/about/page.tsx`
- 简介：找到第一个 `section`，直接改引号里的文字（zh 是中文，en 是英文）
- 工作经历：修改 `workExperience` 数组，按格式增删
- 实习经历：修改 `internship` 数组
- 教育背景：修改 `education` 数组

---

## 二、论文管理

### 增删改论文
文件：`content/papers.md`（**只需编辑这一个文件**）

新增一篇：
```yaml
  - title: "论文标题"
    authors: "Ningning Zhang, ..."
    venue: "期刊名"
    year: 2026
    abstract: "摘要（可为空）"
    doi: "https://doi.org/10.xxx"   # 没有就删掉这行
    pdf: ""                          # 没有就留空
```
保存后刷新浏览器即可，无需重启。

---

## 三、博客文章

### 新建文章
在 `content/blog/zh/`（中文）或 `content/blog/en/`（英文）新建 `.mdx` 文件：
```
文件名建议：2025-08-主题词.mdx
```

文件内容格式：
```markdown
---
title: "文章标题"
date: "2025-08-15"
tags: ["统计学", "时间序列"]
summary: "一句话摘要，显示在列表页"
---

正文内容，支持标准 Markdown 语法。

## 二级标题

代码块、**加粗**、*斜体* 都支持。
```

> ⚠️ 新建文件后需重启服务（关闭终端重新双击脚本）

### 删除文章
直接删除对应的 `.mdx` 文件，重启服务即可。

---

## 四、生活记录（Life）

### 新建记录
在 `content/life/` 新建 `.md` 文件：
```
文件名建议：2025-08-五一.md
```

文件格式：
```markdown
---
title: "五一成都行"
date: "2025-05-03"
tags: ["五一", "旅行", "美食"]
images:
  - /images/life/chengdu-01.jpg
  - /images/life/chengdu-02.jpg
---

正文内容...
```

### 添加照片
1. 把照片放到 `public/images/life/` 文件夹（没有就新建）
2. 在 `.md` 文件的 `images:` 里填写路径：
```yaml
images:
  - /images/life/照片文件名.jpg
```
路径以 `/images/life/` 开头，对应 `public/images/life/` 文件夹。

---

## 五、头像

把头像图片放到 `public/images/avatar.jpg`
然后打开 `components/Sidebar.tsx`，找到：
```tsx
人
```
替换为：
```tsx
<img src="/images/avatar.jpg" className="w-full h-full object-cover rounded-full" alt="avatar" />
```

---

## 六、什么时候需要重启服务

| 操作 | 是否需要重启 |
|------|------------|
| 修改 `.md` / `.mdx` 文件内容 | ❌ 刷新浏览器即可 |
| **新建**或**删除** `.md` 文件 | ✅ 需要重启 |
| 修改 `.tsx` 代码文件 | ❌ 自动刷新 |
| 电脑重启后 | ✅ 需要重新启动 |
